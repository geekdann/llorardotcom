# Audio — llorar.com

Coloca aquí tu archivo de audio. El sitio lo busca en esta ruta:

```
audio/theme.mp3
audio/theme.ogg  (opcional, fallback)
```

## Recomendaciones
- Duración: bucle de 30–90 segundos ideal
- Volumen normalizado: entre -14 y -12 LUFS para no molestar
- Estilo sugerido: 8-bit triste / lo-fi melancólico / música de Game Over
- Formato: MP3 (principal) + OGG (fallback para Firefox)

## Autoplay
El audio intentará reproducirse automáticamente al cargar la página.
Si el navegador lo bloquea (comportamiento normal en Chrome/Firefox),
se activará tras el primer click/tap del usuario.

El botón 🔇 en la esquina superior derecha siempre permite al usuario
activar o silenciar la música.
